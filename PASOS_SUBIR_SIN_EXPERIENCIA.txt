SUBIR EL DASHBOARD - PASO A PASO (sin saber Git)
================================================

YA TIENE GIT instalado en: C:\Program Files\Git\
Solo falta configurarlo una vez y conectar GitHub + Render.

--- PASO A: Configurar Git (5 minutos, una sola vez) ---

1. Doble clic: CONFIGURAR_GIT.bat
2. Escriba su nombre y el mismo correo que usara en GitHub
3. Listo.

--- PASO B: Cuenta GitHub (gratis) ---

1. Entre en https://github.com/signup
2. Cree cuenta y verifique el correo
3. Anote su usuario (ejemplo: favio-campos)

--- PASO C: Subir el codigo ---

1. Doble clic: SUBIR_A_LA_WEB.bat
2. Si pregunta instalar "GitHub CLI", acepte (facilita todo)
3. Cuando pida login GitHub, siga el navegador:
   - gh auth login  (si abre terminal, elija GitHub.com, HTTPS, Login con navegador)

Si NO instala GitHub CLI, haga esto manualmente:

  a) En https://github.com/new cree repositorio:
     Nombre: acr-dashboard
     Publico
     SIN marcar "Add README"

  b) Abra CMD en la carpeta del proyecto y pegue (cambie SU_USUARIO):

     cd "C:\Users\Favio Campos Rivera\Desktop\ACR_dashboard"
     set PATH=C:\Program Files\Git\cmd;%PATH%
     git remote add origin https://github.com/SU_USUARIO/acr-dashboard.git
     git push -u origin main

  c) GitHub pedira usuario y contraseña:
     - Usuario: su usuario GitHub
     - Contrasena: use un TOKEN (no la clave normal)
       Crear token: GitHub - Settings - Developer settings -
       Personal access tokens - Generate (classic) - marque "repo"

--- PASO D: Publicar en internet (Render) ---

1. https://dashboard.render.com/signup  (puede entrar con cuenta GitHub)
2. New  -  Blueprint
3. Conecte el repositorio acr-dashboard
4. Espere 5-15 minutos (plan gratis)
5. Abra la URL que le da (ej: https://acr-dashboard.onrender.com)

--- SI ALGO FALLA ---

- "git no reconocido": use CONFIGURAR_GIT.bat primero
- Push muy lento o falla: instale Git LFS desde https://git-lfs.com
- Pagina en blanco en Render: revise Logs en Render; debe decir "Uvicorn running"

Cuando tenga la URL de Render, pruebela en el celular y en otra PC.
