DASHBOARD ACR - COMO ABRIR EN SU PC
====================================

VERSION WEB (Python):
  1. Instale Python 3.11+  https://www.python.org/downloads/
     (marque "Add python.exe to PATH")
  2. Una sola vez (necesita R + paquete sf):
     Doble clic: exportar_datos.bat
  3. Doble clic: iniciar.bat
  4. Espere en la ventana negra:
     "Abriendo navegador: http://127.0.0.1:8000"
     y "Uvicorn running on http://127.0.0.1:8000"
  5. Si el navegador dice "no se puede conectar":
     - NO cierre la ventana negra hasta ver "Uvicorn running"
     - Abra manualmente: http://127.0.0.1:8000
     - Revise server_log.txt en esta carpeta

Si iniciar.bat no abre ventana:
  - Doble clic: INICIAR.vbs
  - O clic derecho INICIAR_DESDE_POWERSHELL.ps1 -> Ejecutar con PowerShell

VERSION R SHINY (como shinyapps.io):
  - Doble clic: iniciar_shiny.R.bat
  - http://127.0.0.1:8080
