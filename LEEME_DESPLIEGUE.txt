SUBIR EL DASHBOARD A INTERNET — GUIA RAPIDA
==========================================

OPCION RECOMENDADA (1 clic en su PC + 1 clic en Render):

  1. Ejecute exportar_datos.bat   (si aun no lo hizo)
  2. Doble clic: SUBIR_A_LA_WEB.bat
  3. Siga las instrucciones en pantalla
  4. En https://render.com -> New -> Blueprint -> conecte su repo GitHub

URL publica (ejemplo): https://acr-dashboard.onrender.com

---

REQUISITOS EN SU PC:
  - Git: https://git-scm.com/download/win
  - Git LFS (recomendado, mapas ~110 MB): https://git-lfs.com
  - GitHub CLI opcional (automatiza todo): https://cli.github.com
    Luego: gh auth login

---

SI NO TIENE GitHub CLI (subida manual):

  1. Cree repositorio en https://github.com/new  nombre: acr-dashboard
  2. Abra CMD en esta carpeta y ejecute:

     git init
     git branch -M main
     git add -A
     git commit -m "Dashboard ACR web"
     git remote add origin https://github.com/SU_USUARIO/acr-dashboard.git
     git push -u origin main

  3. Render.com -> Blueprint -> seleccione el repo

---

ARCHIVOS QUE DEBEN ESTAR EN GITHUB (no omitir):
  data/cache/centroides_deforestacion.csv
  data/cache/geojson/acr_all.geojson (y zi, limites)
  www/mapas/  (PDF e imagenes de Reportes)
  backend_python/
  frontend/
  render.yaml

NO subir: carpeta .venv (esta en .gitignore)
